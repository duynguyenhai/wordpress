Fast , Flexible Social Jquery based social share bar!

== Description ==

Flat Floating social bar is very fast loaded social share bar . No extra customization needed for this plugin to work. Just install - activate and forget. This plugin will appear every page of your site . 

Demo of this plugin --> http://plugins.wpflaty.com/flat-floating-social-bar/

== Installation ==

1. Upload this plugin in your server ` /wp-content/plugins/ `
2. Activate the plugin through the 'Plugins' menu in WordPress
